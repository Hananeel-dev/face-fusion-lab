Privacy

Inputs are processed to generate a composite, not kept as a gallery.

The app is designed as a small technical demo. It accepts two uploaded photos, creates one blended output, and returns visual similarity metrics.

Back to App

What is processed

The service decodes each uploaded image, detects one face, estimates landmarks, aligns the faces, creates the composite, and calculates metric values.

What is retained

The generated composite image is written to runtime storage so the web page can display it. Runtime outputs are ignored by Git and are not part of the source repository.

What is not intentionally retained

The uploaded input images and individual extracted face crops are not saved by the application code.

About the score

The similarity score is based on visual alignment and pixel variation. It should not be used for authentication, identity verification, or sensitive decisions.
